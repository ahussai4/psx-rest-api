\# PSX REST API



A simple REST API for fetching historical Pakistan Stock Exchange data.



This project is built with FastAPI.



\## Features



\- Fetch historical OHLCV data for PSX symbols

\- Filter data by start date and end date

\- Limit the number of returned rows

\- Sort data in ascending or descending date order

\- Health-check endpoint



OHLCV means Open, High, Low, Close, and Volume.
